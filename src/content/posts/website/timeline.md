> 官方网站：[https://gallery.timeline.ink/](https://gallery.timeline.ink/)


干净、舒适、流畅的多平台壁纸应用。

## 特性
- 跨平台多系统支持（[网页版](https://gallery.timeline.ink/)、[桌面版](https://timeline.ink/)、移动版(拾光壁纸mini小程序)等）
- 海量壁纸开放（收录超5万张，并持续更新中）
- 人工审图（审查后开放，确保内容健康、高质量）
- 原图浏览（不压缩图片，确保刷图体验）
- 原图下载（消耗点数，随意赞助、签到或看广告获赠点数）
- 多图源切换（内置必应等 30+ 源）
- 专题图集（精选“宫崎骏”等 100+ 专题）
- 云收藏（绑定账号，多平台多设备同步）
- 专属推荐（使用越多，推荐越好）
- 本地图库支持（安装桌面版支持）
- 随意设置桌面/锁屏（安装桌面版支持）
- 自动更换桌面/锁屏（安装桌面版支持）
- 每日推送新壁纸（安装桌面版支持）
- 图库 API 合作开放（查看）


拾光项目现阶段由个人开发者运营，且开发仍未停止（多平台支持），考虑到日益增长的成本问题，正在寻求一些合作。如需调用拾光壁纸 API 接口，或其他相关合作需求，可以联系作者，详见：https://doc.timeline.ink/#/about
